# Integrations

Jobify is designed to be extensible. Below are the community-supported and official integrations that enhance Jobify's capabilities.

## [Dishka](https://github.com/reagento/dishka)

[dishka-jobify](https://github.com/C3EQUALZz/dishka-jobify), a Python dependency injection container, seamlessly integrates with [Dishka](https://github.com/reagento/dishka).
